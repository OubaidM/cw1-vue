# CW1 – After-School Classes Web App

## Repositories
- Vue front-end: https://github.com/OubaidM/cw1-vue  
- Express back-end: https://github.com/OubaidM/cw1-api  

## Live Links
- GitHub Pages (Vue app): https://OubaidM.github.io/cw1-vue/  
- Render API (GET /lessons): https://cw1-api.onrender.com/lessons  

## Features
- Browse / sort / search lessons  
- Add to cart & checkout  
- MongoDB Atlas + Express REST API  
- Deployed on GitHub Pages & Render
